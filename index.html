Deottorisierer

Ein user script um die AStA-Website zu verschönern

Version 1.2 Installieren

Über das Projekt

Der AStA Göttingen 2018/19 von ADF, RCDS, LHG hat als erste Amtshandlung versucht das Design ihrer Website zu verändern. Innerhalb von zwei Wochen ist ihnen das gelungen. Nun dürfen alle Wesen eine hässlige AStA-Website betrachten.
Das hier zur Verfügung gestellte user script verschönert die Seite für dich wieder.
Einige Seiten der AStA-Website sehen fehlerhaft aus. Das liegt nicht an diesem Script, sondern daran, dass die Website selbst Fehler enthält.

Installationsanleitung

  1. Einen internetfähigen Browser benutzen (also nicht IE)
  2. Das Addon Greasemonkey (bei Firefox) installieren
    Andere Browser haben ähnliche Addons.
  3. Oben auf den Installationsbutton klicken

Fragen

Falls du Fragen oder Anmerkungen hast, kannst du mir eine Mail mit dem Betreff ADF ist doof schreiben.
